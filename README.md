Visando combater o acesso limitado a materiais em braille, este projeto, desenvolvido como meu Trabalho de Conclusão de Curso (TCC), apresenta um sistema de tradução automática bidirecional (texto-braille e braille-texto) que funciona a partir de imagens ou documentos. O backend, desenvolvido em Django, centraliza o processamento e utiliza duas técnicas principais: um modelo de detecção de objetos YOLOv11 para o reconhecimento automático de células braille e o Tesseract OCR para a extração de texto impresso. O objetivo é fornecer uma ferramenta que facilita a produção de materiais acessíveis, reduzindo os custos e o tempo de tradução manual.
